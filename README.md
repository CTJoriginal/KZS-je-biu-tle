# KŽŠ je biu tle!
Na vikendu Športa in zabave, ki se je odvijal začetek julija 2025, smo v svet spustili drugo serijo KŽŠ nalepk. Ta zemljevid prikazuje vse (znane) lokacije le teh!

Če ste tudi sami prilepili nalepko, lahko sliko le te pripnete v nov [issue](https://github.com/CTJoriginal/KZS-je-biu-tle/issues), ali pa jo pošljete na kontakte navedene v aplikaciji.

Zemljevid je dostopen tukaj: https://ctjoriginal.github.io/KZS-je-biu-tle/
